# vue_node_mall

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

```

## 技术栈
- MVVM框架： Vue 2.0
- 源码：es6
- 构建工具：webpack
- 前端路由：vue-router
- 状态管理：vuex
- 服务端通讯：axios
- 服务端：node.js
- 数据库：mongoDB
